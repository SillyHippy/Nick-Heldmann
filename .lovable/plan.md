

# Portfolio Website for Nick Heldmann

A modern, single-page portfolio site with professional styling and touches of personality — designed to help Nick stand out to employers.

## Design Direction
- **Professional with personality**: Clean layout with a warm color palette (not cold corporate), smooth scroll animations, and friendly typography
- **Initials avatar** in the hero area instead of a photo
- **Single-page scrolling site** with a sticky navigation bar

## Sections (top to bottom)

### 1. Hero / Introduction
- Nick's name prominently displayed with initials avatar
- Objective statement as a brief tagline
- Contact info (email, phone) with clickable links
- **"Download Resume" button** (links to the uploaded PDF)

### 2. Experience
- Timeline-style layout showing each role with company, dates, and description
- Visually distinct cards for each position

### 3. Education
- Western Washington University, Shoreline Community College, and Ballard High School
- GPA and notable activities listed

### 4. Skills
- Clean grid or tag-based layout showing all skills
- Grouped logically (Tech Tools, Soft Skills, etc.)

### 5. Achievements
- Eagle Scout highlight with description
- Certifications and volunteer work listed cleanly

### 6. Footer
- Contact info repeated, downloadable resume link, and simple copyright

## Technical Notes
- Fully static site — no backend needed, deploys directly to Netlify or Cloudflare Pages
- The resume PDF will be placed in the `public` folder for direct download
- Smooth scroll navigation between sections
- Fully responsive (mobile-friendly)
